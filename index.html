SOUNDTREK · Demo

Software development: Martin Gasser

“Soundtrek” was conceived and realised for the sound art festival „Klangmanifeste 2021“. The web based generative composition uses the GPS data of the visitors‘ cell phone to create an individual soundscape which corresponds to the changing atmospheres and moods of the urban environment.

In this demo version you can simulate the walk by moving the position marker on the map.
